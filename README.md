# epd4in26_gray
Demo code to show grayscale for 4.26inch E-Paper from Waveshare and GDEQ0426T82
![image](/images/20240206_233135.jpg "image") 

# Color definition

| Color | Reg 0x24 | Reg 0x26
| ----------- | ----------- | ---------|
| White | 0 | 0 |
| Light | 1 | 0 |
| Dark  | 0 | 1 |
| Black | 1 | 1 |
